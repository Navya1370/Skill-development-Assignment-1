# 🎓 AI-Based Student Productivity Assistant

## 📌 Project Overview

The AI-Based Student Productivity Assistant is a Generative AI web application designed to help students manage their studies efficiently. The application generates personalized study plans, time management tips, and motivational messages based on user input such as subject, topics, deadline, study hours, and study goals.

This project is developed using Python and the Gemini API. It demonstrates the concept of prompt-based content generation in Generative AI by producing dynamic and intelligent responses according to the user's requirements.

---

## 🚀 Features

* Generates personalized study plans
* Provides day-wise schedules
* Gives time management tips
* Generates motivational messages
* Interactive and user-friendly interface
* Real-time AI-generated responses

---

## 💡 Problem Solved

Many students struggle with time management, planning their studies, and organizing academic tasks effectively. This application helps students create structured study schedules and manage deadlines efficiently, reducing stress and improving productivity.

---

## ⚙️ How the Application Works

1. The user enters:

   * Subject
   * Topics
   * Deadline
   * Daily study hours
   * Goal of study

2. The application constructs a structured AI prompt.

3. The prompt is sent to the Gemini language model.

4. The AI generates:

   * Study plan
   * Time management tips
   * Motivational message

5. The generated response is displayed to the user.
